# Week 2 - Regression Modelling

This folder contains the Week 2 regression modelling assignment for the ADA Global Academy MicroMasters portfolio.

Main deliverables are in:

```text
assignment/
```

The assignment includes:

- Multiple linear regression for USA Housing price prediction.
- Logistic regression for diabetes readmission modelling.
- Exploratory data analysis figures.
- Regression coefficients, confidence intervals, odds ratios, diagnostics, and evaluation metrics.
- A reproducible PDF report generated from `Sprint02_BasilEmeokoro.R`.

Raw datasets are not committed. See `assignment/BUILD.md` for reproduction instructions.
