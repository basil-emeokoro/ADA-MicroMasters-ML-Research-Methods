# Agent Notes

This repository is the permanent monorepo for the full 12-week ADA Global Academy MicroMasters portfolio. Future coding agents should read `PROJECT_NOTES.md` before making structural changes.

## Release Tagging

Use annotated tags to mark completed weekly milestones:

```text
v0.1 = Week 1 completed
v0.2 = Week 2 completed
v0.3 = Week 3 completed
v0.4 = Week 4 completed
v0.5 = Week 5 completed
v0.6 = Week 6 completed
v0.7 = Week 7 completed
v0.8 = Week 8 completed
v0.9 = Week 9 completed
v0.10 = Week 10 completed
v0.11 = Week 11 completed
v0.12 = Week 12 completed
v1.0 = Final 12-week portfolio completed
```

Example:

```bash
git tag -a v0.1 -m "Week 1 EDA assignment completed"
git push origin v0.1
```
