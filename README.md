ADA Global Academy MicroMasters – Machine Learning & Research Methods
This repository documents my work throughout the 12-week ADA Global Academy MicroMasters in Data Science, Artificial Intelligence, Machine Learning, and Research Methods.
The objective of this repository extends beyond completing weekly assignments. It serves as a research portfolio demonstrating reproducible data science workflows, statistical analysis, machine learning, software engineering best practices, and scientific reporting.
Repository Objectives
•	Develop research-quality data science projects.
•	Apply statistical and machine learning techniques to real-world datasets.
•	Follow reproducible research principles.
•	Maintain clean software engineering practices using Git and GitHub.
•	Build a portfolio suitable for academic, research, and industry opportunities.
