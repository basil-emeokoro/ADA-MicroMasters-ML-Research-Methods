# Week 1 - Exploratory Data Analysis

Tag: `v0.1`

## Summary

This release marks completion of Week 1 of the ADA MicroMasters ML Research Methods portfolio. The Week 1 assignment performs an exploratory data analysis of Nigeria DHS child nutrition indicators, focusing on weight-for-age z-score (WAZ), weight-for-height z-score (WHZ), and comparison with height-for-age/stunting evidence.

## Highlights

- Completed Nigeria DHS child nutrition EDA using child recode data.
- Implemented DHS anthropometry cleaning rules, including conversion of invalid `9996+` codes to missing values.
- Recoded demographic and socioeconomic variables including breastfeeding status, father education, religion, wealth, and maternal education.
- Produced statistical summaries, missingness checks, subgroup prevalence tables, outlier decision tables, and correlation outputs.
- Generated publication-quality figures for distributional, subgroup, correlation, and outlier analysis.
- Added three research questions grounded in observed Week 1 findings.
- Produced a reproducible PDF report and supporting CSV/PNG outputs.
- Added a GitHub Actions workflow to validate that the Week 1 R script runs successfully.
- Added repository-level metadata and documentation for portfolio readiness.

## Analysis Scope

The Week 1 analysis focuses on:

- WAZ and WHZ cleaning and interpretation.
- National underweight and wasting prevalence.
- Severe underweight and severe wasting prevalence.
- Comparison with HAZ/stunting evidence from class work.
- Subgroup differences by region, residence, wealth, education, breastfeeding, religion, and household context.
- Spearman correlation screening across anthropometric and socioeconomic variables.
- Outlier detection for both WAZ and WHZ.

## Reproducibility

The R workflow is designed to run from:

- repository root;
- `week01-eda/assignment/`;
- RStudio Source;
- command-line `Rscript`;
- GitHub Actions using a synthetic fixture.

The restricted raw DHS dataset is not committed. Local reproduction requires placing the approved DHS child recode CSV in the documented dataset location.
