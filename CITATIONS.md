## Maintenance Notes

- Add each new dataset before or during analysis, not after publication.
- Record usage restrictions clearly enough that a future reviewer can reproduce the workflow without violating data licences.
- Keep package entries focused on major dependencies, not every transitive library.
- Update this file whenever a weekly report introduces a new external resource.
